Reports invalid GDShader declarations.

This inspection checks declaration-level rules such as duplicate symbols, a missing or misplaced shader_type declaration, invalid include paths, empty structs, repeated array sizes, and initializer type mismatches. 

uniform float speed;
uniform float speed; // duplicate declaration

uniform float[3] values[2]; // array size is already defined

#include "res://helpers.gdshader" // only .gdshaderinc files can be included
